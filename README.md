# 虛擬貨幣六脈神劍選幣系統｜CSV 同步區間預估版

本版在「多週期定時刷新版」上新增兩個重點功能：

1. CSV 載入後可加入可刷新幣池，後續定時刷新會用 Binance 即時 K 線更新同幣種。
2. 新增「區間預估」分頁，可查看單支幣在 1m / 15m / 1h / 4h / 1d 的短線上限、短線下限、支撐、壓力、波段上緣與波段下緣。

> 本工具是技術指標與風險輔助，不是投資建議，也不保證預測價格。

## CSV 匯入與同步更新

CSV 格式：

```csv
symbol,base,quote,category,interval,date,open,high,low,close,volume,quote_volume,trades
BTCUSDT,BTC,USDT,主流幣,1h,2026-06-30T00:00:00Z,60000,60500,59800,60300,123.4,7400000,12345
```

使用方式：

1. 用 `download_crypto_multitimeframe.bat` 下載 `crypto_history_multi.csv`。
2. 開啟 HTML，選擇 CSV 或貼入 CSV 文字。
3. 保持勾選「CSV 載入後加入可刷新幣池」。
4. 按「用 CSV 資料選幣」。
5. 系統會把 CSV 內的交易對自動填入幣池。
6. 啟動定時刷新後，會用這批交易對向 Binance 抓最新 K 線，並重新計算多空機率、藏鋒榜、持倉追蹤與區間預估。

注意：CSV 是初始資料；定時刷新會以目前幣池重新抓取所選週期的最新 K 線。

## 區間預估邏輯

單幣區間預估使用三種資料：

1. ATR14 波動區間
2. 近 20 根 K 線高低點
3. 多空機率修正

大致公式：

```text
短線上限 = 現價 + ATR14 × 修正倍率
短線下限 = 現價 - ATR14 × 修正倍率
壓力 = 近20根高點
支撐 = 近20根低點
波段上緣 = max(近20根高點, 現價 + ATR14 × 更高倍率)
波段下緣 = min(近20根低點, 現價 - ATR14 × 更高倍率)
```

修正邏輯：

- 多方機率 >= 70%：上緣放寬，下緣略收斂
- 空方機率 >= 70%：下緣放寬，上緣略收斂
- 多空打架：採保守震盪區間

## 建議刷新設定

- 持倉追蹤：30 秒
- 藏鋒榜：60 秒
- 全幣池掃描：5～15 分鐘

大量幣種與多週期同時刷新會增加 API 請求量，建議全幣池不要用 10 秒激進模式。

## 本機啟動

建議用 ZIP 內的：

```text
start_local_server.bat
```

再用瀏覽器開啟 localhost 頁面，會比直接雙擊 HTML 穩定。
